# batch-rgb-color

## 说明
> 动态修改当前字体颜色槽为指定的RGB颜色


## 调用参数
* R G B [-d default]
* R - 红色代码[0-255], 无则自动生成随机值
* G - 绿色代码[0-255], 无则自动生成随机值
* B - 蓝色代码[0-255], 无则自动生成随机值
* default - 字体颜色重置为原始颜色


## 注意
* 由于default参数是flag型参数，因此必须放到参数列表最后
